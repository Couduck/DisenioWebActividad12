¿Qué es lo que aprenderás en este curso?

Bootstrap

Una librería bastante poderosa y documentada para el desarrollo de Front-end de facil instalación, uso y customización propia de los elementos que maneja. Además, es multilenguaje, por lo que puedes implementarlos en varios proyectos web que funcionen con PHP, Java, etc..

Vue.js

Herramienta basada en javascript que permite de forma muy simple realizar parte del Back-end y el Front-End desde un mismo archivo, en conjunto con Bootstrap es capaz de realizar páginas web funcionales y visualmente atrayentes para los internautas.

GitHub

Herramienta de control de versión de programas que nos permitirá crear diferentes puntos de guardado de un mismo programa tanto dentro de nuestra computadora como en almacenamiento en nube. Es una de las herramientas mas utilizadas por desarrolladores, tanto que se considera como una obligación para el perfil profesional de los mismos si es que se desea tener una oportunidad en el mercado laboral
